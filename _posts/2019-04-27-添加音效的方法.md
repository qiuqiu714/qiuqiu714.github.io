# 一. 直接播放音效
先在对应脚本中添加实例
```
public AudioClip tankExplosionAudio;
```
不能忘记在Inspector里赋值 
在对应位置添加播放代码即可
```
AudioSource.PlayClipAtPoint(bulletExplosionAudio,transform.position);
```
