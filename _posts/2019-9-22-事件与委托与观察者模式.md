# 一. 委托
## 1.一般委托
### 定义委托类型
```
public delegate 返回值 delegateName(参数);
```
### 定义委托
```
delegateName a;
```
### 赋予方法
```
a = 方法名;
```
### 调用
```
a();
```

## 1.Action委托
Action委托是无返回值，可有参数的委托
### 定义委托(无返回值的方法)
```
Action<参数类型> a;
```
### 赋予方法
```
a = 方法名;
```
### 调用
```
a();
```

## 1.Func委托
Func委托是有返回值，可有参数的委托
### 定义委托
```
Func<参数类型，参数类型，....返回值类型> a;
```
### 赋予方法
```
a = 方法名;
```
### 调用
```
a();
```



# 二. 事件
在定义事件前需要先定义委托，再定义事件
```
//定义一个委托
public delegate void delegateRun();
//定义一个事件
public event delegateRun eventRun;
//
judgment.eventRun+=new Judgment.delegateRun(runsport.Run);
```
